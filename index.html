Pre-work Phase 1

Git (on my computer) is version control software that we run on our computer.

GitHub (on the web) is a REMOTE web application that works with Git to help us collaborate and share our code with others.

ddc-web-student.cnm.edu (the server) is the bootcamp SERVER that runs our code.

We add and commit our code to Git on our computer

We PUSH our code from our computer to GitHub.

We DEPLOY our code from our computer to the bootcamp server.

DEPLOY and PUSH are different!
